解說
Cloud SQL はフルマネージドのリレーショナルデータベースサービスで、
PostgreSQL に完全対応しています。 アプリケーションを大幅に変更するこ
となく、既存の SQL ロジックやトランザクションをそのまま活用でき、
ACID 特性や強整合性が求められるユースケースにも最適です。

クション処理やリレーション管理には不適です。
b. Dataproc: Apache Hadoop や Spark 向けのサービスで、 トランザクション処
理ではなくビッグデータバッチ処理に特化しています。
>
