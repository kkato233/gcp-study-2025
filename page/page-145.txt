解説
Windows VM の作成時にメタデータに windows-password キーを追加するこ
とで、指定ユーザーのパスワードを事前に定義できます。 これにより RDP
ログインが可能となり、マニュアル操作なしにユーザーのログイン環境が整
います。
不正解の解説
a. Active Directory 連携は大規模環境向けで初期構築が複雑すぎます。
b. IAP は Web アプリケーションの認証には有効ですが、 RDP 接続の直接認
証には対応しません。
c. Startup Script によるコマンド実行は可能ですが、タイミングやエラー処理
が難しく、確実性に欠けます。
