解説

OS の脆較性情報を正確に取得するには、各 Compute Engine インスタンスに
OS Config エージェントをインストールし、セキュリティ担当者に
roles/osconfig.vulnerabilityReportViewer を付与するのが最適です。これにより、
Google Cloud の脆能性レポート機能を利用して、セキュリティ関連のインサ
イトを GUI または API 経由で一元管理できます。
