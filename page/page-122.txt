問題 10

南極観測基地のデータ分析用インフラでは、センター内の Linux 仮想マシン
(VM) から、GCP の Cloud SQL インスタンスに接続し観測データを鞭積し
ています。この際、VM に対して特定のアクセス権をもつカスタムサービス
アカウントを適用し、API アクセスを制御する必要があります。どの設定方
法が最も適切ですか ?

a. カスタムサービスアカウントの JSON キーを GCS バケットにアップロー
BU, VM 起動時に読み込ませる

b. サービスアカウントを IAM で VM グループに紐づけるスクリプトを用意
して後付けで適用する

c. VM 作成時に「ID と API アクセス」でカスタムサービスアカウントを指
定する

d. JSON キーを Secrets Manager に登録し、VM から API 経由で読み取って認
証する
