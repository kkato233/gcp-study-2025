解説

VM 作成時に「ID と API アクセス」セクションでサービスアカウントを指
定することで、VM がそのアカウントの権限で Cloud SOL などのリソースに
安全かつ自動的にアクセスできます。JSON キーの管理も不要となり、セ
キュアな構成です。

不正解の解説

a. GCS 経由でのキー読み込みは手動操作が必要で、セキュリティ上のリス
クも伴います。

因になりやすいです。

d. Secrets Manager は機密情報管理に適しますが、VM からのキー取得手順は
複雑で推奨されません。
