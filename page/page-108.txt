あなたは、ある小売系スタートアップのeコマース基盤を gcloud CLI で操作
する担当者です。 社内プロキシの使用が必須ですが、 セキュリティ要件とし
てプロキシ認証情報 (ユーザー名とパスワード)をログに出力させたくあり
ません。安全に設定するにはどうすべきでしょうか?
a. gcloud config set proxy/username & gcloud config set proxy/password
を使って設定ファイルに認証情報を保存する
b. 認証情報を SHA-256 でハッシュ化し、 証明書用設定ファイルに保存して
参照させる
c. システムの環境変数として CLOUDSDK_PROXY_USERNAME および
CLOUDSDK_PROXY_PASSWORD を設定する
d. 認証情報を直接 ~/.config/gcloud/configurations/config_default に手書きする
